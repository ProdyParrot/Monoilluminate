# Monoilluminate

A simple mobile puzzle game created in a 2 dau [eropd for the Bushiroad × DigiPen (Singapore) Mobile Game Jam 2019.

**Solo Project** – Unity 2019

## About the Project
Monoilluminate is a minimalist puzzle game where players manipulate light rays to solve increasingly complex levels. Every puzzle, game mechanic, and piece of UI was designed and coded from scratch within 2 days.

The game received the **AppLovin Special Reward** (a side award outside of the top 3 placements).

## Game Jam Details
- **Event**: Bushiroad × DigiPen Mobile Game Jam 2019
- **Duration**: 26 hours
- **Role**: Designer / Programmer (Solo)
- **Engine**: Unity 2018
- **Target Platform**: Mobile

## Features
- Complete puzzle system with multiple mechanics
- Hand-crafted levels with progressive difficulty
- Clean, minimal UI designed for touch controls
- Full game flow (menu → gameplay → win/lose screens)
- Optimised for mobile performance

## How to Run
1. Open the project in Unity 2018.3
2. Load the scene 'Title.unity'
3. Press Play

The game is fully playable and was built to run smoothly on mobile devices.

## Technical Highlights
- All puzzles and mechanics built from scratch in C#
- Touch-friendly input system
- Simple but effective level data structure
